# Esprokurt

Rede social interna inspirada no Orkut, com visual moderno, criada como projeto do programa **Gestores do Futuro** (ESPRO).

## Contexto

Este projeto foi desenvolvido com foco em:

- desenvolvimento de visão de produto
- colaboração em equipe
- aplicação prática de HTML, CSS e JavaScript
- criação de soluções com identidade visual forte

A proposta do Esprokurt é unir:

- **nostalgia do Orkut** (estrutura social, comunidade, perfil, mensagens)
- **estética moderna** (tema escuro/claro, layout limpo, microinterações e responsividade)

## Objetivo do Projeto

Criar um site interno de rede social para simular conexões entre pessoas, comunidades e conteúdos, reforçando conceitos de produto digital e experiência do usuário.

## Funcionalidades

- feed com publicações
- perfil de usuário
- página de amigos
- comunidades
- mensagens
- notificações com filtros
- configurações
- tema claro/escuro com persistência
- tela de carregamento animada
- avatares com placeholders em emoji (sem dependências externas)

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)

## Identidade do Projeto

- tom visual: social, acolhedor, vibrante
- proposta: ambiente interno com experiência leve e intuitiva
- direção de design: moderno com referências nostálgicas
